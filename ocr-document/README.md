# Document Processing with OCR / Computer Vision

โครงการประมวลผลเอกสารอัตโนมัติด้วย OCR และ Computer Vision  
- Business Impact: ลดเวลาในการตรวจสอบ/ยืนยันเอกสารลง 70% (เหมาะสำหรับบริการฐานรากประชาชน)  
- Tech Stack: EasyOCR (รองรับภาษาไทยดี) + Streamlit สำหรับอัปโหลดภาพและแสดงผล  
- Features: Extract ข้อมูลจากบัตรประชาชน, สลิปโอนเงิน, เอกสารสัญญา  
- Status: กำลังเพิ่ม code, ตัวอย่างภาพทดสอบ, deployment demo เร็วๆ นี้  
- Expected deploy: Streamlit Cloud หรือ Render.com
