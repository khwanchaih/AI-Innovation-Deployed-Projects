# Thai LLM Banking Chatbot / Agent

โครงการพัฒนา AI Chatbot/Agent ภาษาไทยสำหรับบริการธนาคาร  
- Business Impact: ลดภาระ Call Center ประมาณ 40% ด้วยการตอบคำถามลูกค้าอัตโนมัติ  
- Tech Stack: LangChain + Hugging Face (โมเดลภาษาไทย) + Streamlit สำหรับ Demo  
- Features: RAG (Retrieval-Augmented Generation) สำหรับดึงข้อมูลจากเอกสารธนาคาร/กฎระเบียบ BOT  
- Status: กำลังเพิ่ม code, ตัวอย่าง prompt, deployment demo เร็วๆ นี้  
- Expected deploy: Streamlit Cloud หรือ Render.com
